### 리눅스 명령어
---
**1) top**  
top 명령어란?  
: 실시간으로 CPU 사용률을 체크해주는 도구, 리눅스를 사용하는 서버의 성능이나 현재 돌아가고 있는 상활을 볼 때 사용

- 명령어 사용법 > **top**  
- 명령어 옵션 (대부분 옵션은 shift를 통한 명령을 주로 사용)  
| 옵션 | 기능 |  
| :----: | :----: |  
| shift + p | CPU 사용률 내림차순 |  
| shift + m | 메모리 사용률 내림차순 |  
| shift + t | 프로세스가 실행되고 있는 시간 순서 |
